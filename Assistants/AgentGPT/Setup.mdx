Welcome to the AgentGPT setup documentation. This guide will assist you in configuring AgentGPT to retrieve and summarize information effectively. Follow the steps below for a successful setup.

## Steps to Setup AgentGPT

### Pre-Requisites
- AgentGPT assistant should be configured. [Click here](../Introduction/getting-started.mdx) to know more about the process of AgentGPT Assistant .

### Step 1: Locate the Agent Tools
- Navigate to the Home page
- Locate the Left Panel
- Click on `Agents` 

### Step 2: Choose the Agent Tool
- Choose from the available tools

### Step 3: Configure the Tool
- Currently the Supported Tools are:

   1. [PostgreSQL](../Agent-Tools/PostgreSQL.mdx)
  
### Step 4: Navigate to PlayGround 
- Navigate to the Home page
- Locate the Left Panel
- Click on `PlayGround` 

## Congratulations! AgentGPT setup is complete. You're all set to start chatting with your agent.
